# 欢迎来到我的博客

这是我的第一篇博客文章！我很兴奋能够分享我的想法和经验。

## 关于这个博客

这个博客是使用以下技术栈构建的：

- **Next.js 15** - React 框架
- **MDX** - 支持 JSX 的 Markdown
- **Tailwind CSS v4** - 样式框架
- **TypeScript** - 类型安全

## 为什么选择 MDX？

MDX 让我可以在 Markdown 中使用 React 组件，这意味着我可以创建更加互动和丰富的内容。

```javascript
// 示例代码块
const greeting = (name) => {
  return `Hello, ${name}!`;
};

console.log(greeting('World'));
```
